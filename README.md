## Installation

```bash
# Clone this repository
$ git clone https://github.com/thurdev/AyxBot

# Go into the repository
$ cd AyxBot

# Install dependencies
$ yarn install

# Don't forget to configure your .env file, just rename the .env.example to .env

# Run the app
$ yarn dev
```
